 Para las imagenes que se muestran dentro de la pagina web se creo un 
repositorio en la raiz del disco duro c:\ la ruta para la misma es la
siguiente C:\CarritoCompras\images\2014\08 es de suma importancia 
copiar la carpeta CarritoCompras en la raiz de C:\ para que la misma 
se pueda referenciar en la aplicacion, dicha carpeta se encuentra en
el mismo folder de este documento.

 Luego se debe de realizar un segundo paso, copiar el archivo 
"CarritoCompras#upload_images.xml" en la siguiente ruta 
%CATALINA_HOME%\conf\Catalina\localhost este archivo tambien esta 
contenido en la carpeta de este documento.
 
 Nota: Estos pason son necesario para que se puedan ver las fotos en la 
aplicacion.
